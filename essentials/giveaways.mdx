## Setting Up Giveaways

<Steps>
  <Step title="Create Giveaway">
    From your dashboard: - Click "Create Giveaway" - Set giveaway name and
    description - Define entry period
  </Step>
  <Step title="Add Prizes">
    Configure prize details: - Prize description - Contributing vendor - Value -
    Quantity
  </Step>
  <Step title="Entry Form">
    Customize information collected: - Name - Email - Phone (optional) - Custom
    questions
  </Step>
  <Step title="Generate QR Codes">
    Create entry points: - Main market QR code - Vendor-specific codes - Social
    media sharing
  </Step>
</Steps>
